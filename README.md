# MaidVoicePitch

Expands builtin slider limits and adds additional sliders.

Requirements:

- AddModsSlider
- ExternalSaveData
- ExternalPresetData

## Installing

Download the latest version from [the release page](../../releases/latest). Extract the archive contents into the root of your game directory.

## Usage

While in the character editor, click AddModsSlider or press F5.
